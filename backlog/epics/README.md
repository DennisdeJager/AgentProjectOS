# Epics

Versiedatum: 2026-05-19 13:34:29 +02:00

Epics beschrijven grote projectdoelen. Een epic is niet direct uitvoerbaar door een agent en moet worden opgesplitst in stories en werkpakketten.

Minimale velden:

- ID
- titel
- businessdoel
- scope
- out of scope
- stories
- risico's
- succescriteria
