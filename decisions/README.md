Gebruik deze map voor architectuur- en procesbesluiten.

Maak een ADR wanneer:

- een structurele architectuurkeuze wordt gemaakt;
- een technology stack wordt gekozen;
- een validatiestrategie wordt vastgelegd;
- een security- of deploymentkeuze blijvende impact heeft.
