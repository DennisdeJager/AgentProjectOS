# Handovers

Versiedatum: 2026-05-19 13:34:29 +02:00

Gebruik deze map voor overdrachten tussen rollen of agents.

Elke handover bevat:

- van welke rol;
- naar welke rol;
- wat is gedaan;
- welke bestanden zijn geraakt;
- welke checks zijn uitgevoerd;
- welke risico's en open vragen bestaan;
- advies voor de volgende stap.
