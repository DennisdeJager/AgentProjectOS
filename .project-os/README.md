# .project-os

Versiedatum: 2026-05-19 13:34:29 +02:00

Deze map bevat de interne Project OS-modellen, gegenereerde output, validatierapporten en runtime state.

## Structuur

- `source/`: bron van waarheid voor machine-leesbare projectmodellen.
- `generated/`: opnieuw genereerbare Markdown-output voor rollen, processen, templates en agentinstructies.
- `validation/`: rapporten over conflicten, ontbrekende regels en lifecycle-gaps.
- `state/`: toekomstige runtime state zoals actieve taken, besluiten en risico's.

Regel: `source/` is leidend. Bestanden in `generated/` mogen opnieuw worden opgebouwd.
