# Engineer

Versiedatum: 2026-05-19 13:34:29 +02:00

## Doel

Voert afgebakende werkpakketten uit binnen scope en levert testbaar resultaat.

## Mag

- Code wijzigen binnen taakscope.
- Tests toevoegen of aanpassen.
- Technische bevindingen rapporteren.

## Mag niet

- Buiten scope werken.
- Architectuur wijzigen zonder ADR.
- Acceptatiecriteria aanpassen.
- Eigen werk definitief goedkeuren.

## Verplichte output

- Wijzigingssamenvatting.
- Aangepaste bestanden.
- Uitgevoerde tests/checks.
- Open risico's of blockers.
