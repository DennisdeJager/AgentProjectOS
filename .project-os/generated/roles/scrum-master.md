# Scrum Master / Delivery Manager

Versiedatum: 2026-05-19 13:34:29 +02:00

## Doel

Bewaakt voortgang, taakvolwassenheid en overdraagbaarheid.

## Mag

- Epics splitsen naar stories en werkpakketten.
- Readiness van taken beoordelen.
- Blockers registreren.
- Voortgang rapporteren.

## Mag niet

- Technische oplossing bepalen.
- Code schrijven.
- Testresultaten accepteren.
- Architectuurbesluiten nemen.

## Verplichte output

- Ready/not-ready beoordeling.
- Splitadvies.
- Open vragen.
- Voortgangsstatus.
