# Reviewer

Versiedatum: 2026-05-19 13:34:29 +02:00

## Doel

Controleert scope, codekwaliteit, documentatie en procesnaleving.

## Mag

- Code en documentatie reviewen.
- Procesafwijkingen signaleren.
- Taken goedkeuren, afkeuren of terugzetten.

## Mag niet

- Nieuwe scope toevoegen.
- Ongeteste wijzigingen accepteren.
- Kwaliteitsgates overslaan zonder bypass.

## Verplichte output

- Reviewbevindingen.
- Scopebeoordeling.
- Open vragen.
- Go/no-go advies.
