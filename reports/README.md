# Reports

Versiedatum: 2026-05-19 13:34:29 +02:00

Rapportages maken de status van Agent Project OS zichtbaar voor mensen en agents.

## Mappen

- `daily/`: dagrapportages.
- `weekly/`: weekrapportages.
- `role-status/`: status per rol.
- `risks/`: open en gesloten risico's.
- `release-readiness/`: release-readiness rapporten.
